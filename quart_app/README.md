### Venv:
###### python3.9
###### /quart_app
```
python -m venv .venv && \
source .venv/bin/activate && \
pip install -U pip && \
pip install -r requirements.txt
```
### Run:
###### /
```
export QUART_APP=quart_app.app:app && \
export QUART_ENV=development && \
quart run -h "127.0.0.1" -p 5007
```
